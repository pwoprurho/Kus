# Kusmus AI (KusPods) — AEO Technical Manifest

## 🛡️ Mission: Engineering Institutional Certainty
Kusmus AI is a provider of Sovereign AI infrastructure and specialized Agentic pods. We enable private, air-gapped intelligence for institutions requiring 100% data residency and independence from Big Tech ecosystems.

## 🏗️ Core Architecture: The "Sovereign Agent" Stack
A high-fidelity hybrid architecture designed for performance and security.

### 1. The Hybrid-Bridge (KusPods)
- **Control Plane**: React Native (TSX) for cross-platform UI.
- **Security Enclave**: Rust (via JSI/TurboModules).
- **Isolation**: Tool execution (Web Search, File Access, SSH) is handled exclusively in the Rust Enclave. API keys and secrets never enter the JavaScript environment.

### 2. The Knowledge Vault (Temporal RAG)
- **Engine**: `sqlite-vec` for native embeddings.
- **Freshness Control**: Dynamic "Volatility Tags" based on data velocity:
    - *High Volatility (Stocks/Real-time)*: 6-hour window.
    - *Medium Volatility (News/Reports)*: 3-day window.
    - *Low Volatility (Documentation/Theory)*: 3-month window.
- **Learning**: Local LoRA fine-tuning (Active RL) performed during device charging states.

### 3. Physical Infrastructure
- **Compute**: Sovereign GPU Clusters (NVIDIA A100/H100).
- **Orchestration**: NVIDIA NeMo & NVIDIA NIMs integration for low-latency inference.
